# Module 08 — Reports & Export

## Overview

Comprehensive report generation system supporting 8 report types with CSV and PDF export. Reports are generated server-side for PDF (PDFKit) and client-side for quick CSV downloads.

---

## Report Types

| Report | Description | Key Metrics |
|---|---|---|
| **Fleet Report** | Overall fleet overview | Total vehicles, utilization, health distribution, status breakdown |
| **Vehicle Report** | Individual vehicle history | Trips, fuel, maintenance, expenses, health trend for one vehicle |
| **Driver Report** | Individual driver performance | Trips, safety score, violations, hours, compliance |
| **Expense Report** | Cost breakdown by category | Category totals, vehicle costs, monthly trends |
| **Fuel Report** | Fuel consumption analysis | Total liters, cost, efficiency, anomalies, per-vehicle breakdown |
| **Maintenance Report** | Maintenance activity summary | Completed, pending, overdue, costs, common issues |
| **ROI Report** | Return on investment per vehicle | Revenue vs costs, profitability, payback period |
| **Trip Report** | Trip operations summary | Volume, completion rate, avg duration, revenue, per-route analysis |

---

## Report Parameters

All reports accept these parameters:

| Parameter | Type | Default |
|---|---|---|
| dateFrom | Date | First of current month |
| dateTo | Date | Today |
| vehicleId | String (optional) | All vehicles |
| driverId | String (optional) | All drivers |
| vehicleType | VehicleType (optional) | All types |
| region | String (optional) | All regions |
| format | 'pdf' \| 'csv' | pdf |

## One-Click PDF Trip Report

### Purpose

Generate a professional, self-contained trip report with a single click. Contains all trip data, embedded map snapshot, QR verification code, and financial breakdown. Designed for printing and client handoff.

### Technology

- **PDF Rendering**: Puppeteer (HTML → PDF for pixel-perfect layout)
- **Map Snapshot**: Puppeteer screenshot of a Leaflet map page with route overlay
- **QR Code**: `qrcode` npm package generates data URL embedded in template
- **Template Engine**: HTML template with CSS Flexbox/Grid layout

### Map Snapshot Generation

```typescript
async function captureMapSnapshot(routeGeoJSON: GeoJSON, source: LatLng, dest: LatLng): Promise<Buffer> {
  const browser = await puppeteer.launch({ headless: true });
  const page = await browser.newPage();
  await page.setViewport({ width: 800, height: 400 });
  
  // Load a minimal HTML page with Leaflet + route
  await page.setContent(mapTemplate(routeGeoJSON, source, dest));
  await page.waitForSelector('.leaflet-tile-loaded'); // Wait for tiles
  await page.waitForTimeout(1000); // Extra buffer for rendering
  
  const screenshot = await page.screenshot({ type: 'png' });
  await browser.close();
  return screenshot;
}
```

## Implementation Notes

- **Server-side PDF**: PDFKit with streaming response (`res.setHeader('Content-Type', 'application/pdf')`)
- **Client-side CSV**: Built from API data, triggers `<a download>` link
- **Report history**: Last 50 reports stored with metadata (type, params, generated by, file URL)
- **Chart embedding**: Server renders charts as PNG using chart.js on canvas (server-side), embeds in PDF
- **Large reports**: Streaming PDF generation to avoid memory spikes on 1000+ row reports
